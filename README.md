# MUAY THAI FAMILY GYM

React + Vite + SCSS landing page.

## Як запустити

```bash
npm install
npm run dev
```

## Зібрати фінальну версію

```bash
npm run build
npm run preview
```

## Що всередині

- адаптивність від 320px до 1920px
- українська / англійська мова
- темний спортивний fight-style дизайн
- блоки: Hero, About, Programs, Schedule, Coaches, Army Support, Contacts
- оптимізовані WebP-зображення
- кнопки на телефон, Instagram, Facebook і Google Maps
